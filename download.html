// TẢI PATCH

Patch Việt Hóa Installer

Phần mềm cài đặt riêng của nhóm Dịch 2000s. Chỉ áp dụng cho những tựa game mà nhóm đã dịch xong — hiện tại là Plague Inc. Phần mềm giúp việc cài đặt bản Việt hóa gọn nhẹ, không cần thao tác thủ công phức tạp.

PatchVietHoaInstaller
Phiên bản mới nhất trên GitHub Releases
Tải về
Công cụ này hoàn toàn miễn phí. Nếu bạn thấy nơi nào yêu cầu trả phí để lấy được phần mềm này, đó chắc chắn là lừa đảo — vui lòng chỉ tải từ trang GitHub chính thức của nhóm bên dưới.
CÁCH SỬ DỤNG
01

Tải công cụ

Bấm nút "Tải về" phía trên để lấy bản cài đặt mới nhất từ GitHub Releases.

02

Mở công cụ

Chạy file cài đặt vừa tải về. Công cụ sẽ hiển thị danh sách các game đã có bản Việt hóa.

03

Chọn game & thư mục cài đặt

Chọn tựa game muốn Việt hóa, trỏ đúng đến thư mục cài đặt game trên máy bạn.

04

Cài đặt & chơi

Bấm cài đặt, đợi công cụ xử lý xong rồi mở game lên chơi bằng tiếng Việt.

MÃ NGUỒN

Công cụ được phát triển mã nguồn mở. Bạn có thể xem toàn bộ mã nguồn, báo lỗi hoặc đóng góp trực tiếp trên GitHub.
